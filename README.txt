# Atlas Projeto

## Descrição
O Atlas é um guia turístico completo que ajuda você a explorar as melhores atrações, restaurantes, locais de compras e hospedagens em várias cidades ao redor do mundo.

## Instalação

1. Clone o repositório:
    git clone https://github.com/usuario/atlas-projeto.git

2. Navegue até o diretório do projeto:
    cd atlas-projeto

3. Instale as dependências:
    npm install

4. Configure o banco de dados MySQL:
    - Crie um banco de dados chamado `atlas`.
    - Importe as tabelas e dados necessários a partir do arquivo `atlas.sql`.

5. Configure as variáveis de ambiente:
    DB_HOST=localhost
    DB_USER=seu_usuario
    DB_PASS=sua_senha
    DB_NAME=atlas

6. Inicie o servidor:
    npm start

## Estrutura do Projeto
- `index.html`: Página principal.
- `styles/`: Diretório de arquivos CSS.
- `scripts/`: Diretório de arquivos JavaScript.
- `images/`: Diretório de imagens.
